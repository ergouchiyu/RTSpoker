# Card RTS - 原型

**一边打扑克，一边实时派兵偷袭对方的手牌**

## 项目简介

Card RTS 是一款创新的混合类型游戏，将实时策略（RTS）与扑克完美结合。

### 核心玩法

- **手牌产资源** - 每张扑克牌持续产生补给
- **出牌推进胜利** - 目标是最先出完所有牌
- **派兵骚扰** - 用步兵攻击对手的牌，降低完整度

### 游戏规则

1. 每个玩家有 10 张手牌
2. 每张牌持续产生补给（资源）
3. 可以消耗补给生产步兵
4. 步兵会自动攻击对手的牌
5. 牌的完整度降低会减少收入
6. 最先出完所有牌的玩家获胜

## 操作说明

### 基本操作

- **点击手牌** - 选中/取消选中
- **点击"步兵"按钮** - 生产步兵（消耗100补给）
- **点击"维修"按钮** - 维修选中的牌
- **点击"结束回合"** - 切换到对手回合

### 游戏流程

1. 点击"开始游戏"或"vs AI"
2. 每回合可以：
   - 出牌（选中牌后自动出牌）
   - 生产步兵
   - 维修牌
3. 点击"结束回合"切换玩家
4. 最先出完所有牌获胜
